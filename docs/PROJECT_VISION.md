# Project Vision

## Problem Statement
Modern data platforms generate large volumes of metadata such as pipeline runs, ingestion logs, and data quality metrics. However, accessing and interpreting this information typically requires writing SQL, navigating dashboards, and manual debugging.

## Vision
Enable users to interact with data systems using natural language.

## Target Users
- Data Engineers
- Data Analysts
- Data Platform Teams

## Core Value
- Reduce dependency on SQL for operational insights
- Improve accessibility of observability data
- Accelerate debugging and decision-making
