# Orion Data Copilot

An AI-powered data copilot that enables natural language interaction with data pipelines, ingestion systems, and data quality metadata.

This project simulates an intelligent interface on top of a modern data platform, allowing users to query pipeline runs, investigate failures, and explore data health using simple natural language queries.

---

## 🚀 Motivation

Modern data platforms generate large amounts of operational metadata:
- pipeline runs
- ingestion logs
- data quality metrics

However, accessing and understanding this information often requires:
- writing SQL queries
- navigating dashboards
- manually inspecting logs

Orion Data Copilot aims to solve this by providing:

> A natural language interface for interacting with data systems.

---

## 🎯 Key Features (Planned)

- Natural language → SQL translation
- Query ingestion run history (IngestFlow)
- Query data quality results (SentinelDQ)
- Pipeline failure analysis and summarization
- Interactive CLI / UI interface
